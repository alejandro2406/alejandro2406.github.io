As impresoras 3D permiten crear obxectos a partir de modelos 3D feitos por ordenador. Funcionan depositando un filamento de plástico derretido nun extrusor, capa por capa.

Pódense usar para crear obxectos personalizados ou prototipos, xa que os diseños se poden crear nun ordenador e imprimir rapidamente. Os diseños pódense crear mediante software de diseño 3D, como Blender ou Autocad, mediante fotogrametría (a partir de fotos dun obxecto real) ou descargar en sitios como Thingiverse.

Algúns dos obxectos que se poden imprimir son chaveiros personalizados, rodas dentadas, esculturas ou pezas de recambio para impresoras 3D.

Entre os usos das impresoras 3D están facer maquetas ou prototipos, obxectos personalizados, recambios para mecanismos ou próteses.
